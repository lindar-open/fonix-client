# fonix-client
JVM client for the [Fonix](https://www.fonix.com/documentation) integration


## Usage: 
### Maven:
```xml
<dependency>
    <groupId>com.lindar</groupId>
    <artifactId>fonix-client</artifactId>
    <version>1.6.0</version>
</dependency>
```
### Gradle:
```kotlin
implementation("com.lindar:fonix-client:1.6.0")
```
